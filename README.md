# stu-mini-app

Мини-приложение Telegram для записи на студию.

## Что умеет

- выбор месяца (на 12 месяцев вперед);
- выбор числа с учетом количества дней в месяце;
- выбор времени (слоты с 09:00 до 21:30);
- подтверждение записи в интерфейсе;
- отправка данных в бота через `Telegram.WebApp.sendData`;
- сохранение последней записи в `localStorage`.

## Запуск локально

```bash
python3 -m http.server 8080
```

Откройте: <http://localhost:8080>

## Файлы

- `index.html` — разметка mini app;
- `styles.css` — стили;
- `app.js` — логика выбора даты/времени и интеграция с Telegram WebApp API.
